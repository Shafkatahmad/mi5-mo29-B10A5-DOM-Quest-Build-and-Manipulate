Important FAQ

What is the Document Object Model (DOM)?
The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.
How do you select an element from DOM?
You can select an element from DOM using 5 ways.
1. getElementByTagName()
2. getElementsByClassName()
3. getElementById()
4. querySelector()
5. querySelectorAll()
What is event delegation in the context of DOM, and why is it important?
Event delegation is a technique in JavaScript where a parent element handles events for its child elements.
Why important: Instead of managing numerous individual listeners, you have a single, centralized listener on the parent, making code easier to understand, modify, and debug.
How do you manipulate an element's attributes and styles using DOM?
You can manipulate an element's attributes and styles by selecting the element and then using specific properties and methods to modify its characteristics.
